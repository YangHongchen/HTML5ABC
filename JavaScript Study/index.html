Javascript 是一门高端的，动态的，弱类型的编程语言。非常适合面向对象和函数式的编程风格。
JavaScript的语法来自Java，他的一等函数来自Scheme，他的基本原型（prototype-baseed)继承来自于Self。

JavaScript支持的数据类型：int,string,char,boolean,null
当然还有两种非常重要的两种类型:对象（Object)和数组(Array)

JavaScript中，对象是由花括号括起来的；（通俗易懂）
注意：JavaScript中的字符串的对比，默认是通过字母表中的索引来排序的； 例如: tw > th = true

逻辑运算符是对布尔值的合并或者取反（精炼的总结）
